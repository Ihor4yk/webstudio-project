<h1>WebStudio</h1>

<h2>Description:</h2>
<p>A fully responsive multi-page website for a web design studio, built using semantic HTML and modular SCSS architecture.</p>

<h2>Technologies:</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3/SCSS</li>
  <li>Responsive layout</li>
  <li>Modern CSS techniques (Flexbox, Grid)</li>
</ul>

<h2>Features:</h2>
<ul>
  <li>Adaptive layout for all screen sizes</li>
  <li>Well-structured SCSS architecture</li>
  <li>Multi-page navigation</li>
  <li>Accessible and semantic markup</li>
</ul>
